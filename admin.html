Setup Admin

Untuk mengedit project tanpa server, admin panel ini menggunakan GitHub API dengan Personal Access Token. Token hanya tersimpan di browser kamu — tidak dikirim ke mana pun.

  1. Buka github.com/settings/tokens/new
  2. Centang scope repo → Scroll bawah → Generate token
  3. Salin token yang muncul (hanya muncul sekali!)
  4. Paste di bawah ini
❌ Token tidak valid atau tidak punya akses ke repo ini. Coba lagi.
